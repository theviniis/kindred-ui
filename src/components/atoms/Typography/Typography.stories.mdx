## <a id="usage"></a>Usage

> ### <a id="usage-children"></a>Passing text as children

You can simple render our `Typography` component passing your text as `children` prop.

```jsx
<Typography type='headline'>This is my header.</Typography>
<Typography type='body'>This is my paragraph.</Typography>
```

<Canvas>
  <Typography type='headline'>This is my header.</Typography>
  <Typography type='body'>This is my paragraph.</Typography>
</Canvas>


> ### <a id="usage-any-component"></a>Styling any component

First you need to create your `Wrapper` extending `Typography` styles.

```jsx
const Wrapper = styled(Typography)``;
```

Them use `.attrs()` method passing an object defining a `typography` `{ type: 'body' }`.

```jsx
const Wrapper = styled(Typography).attrs({ type: 'body' })`
  // your styles goes here
`;
```

Render your Wrapper passing `forwardedAs="element"` prop.

```jsx
<Wrapper forwardedAs="div">I'm an Typography styled div</Wrapper>
```

Your Wrapper will be rendered as a `div` tag and will be styled with `TYPOGRAPHY_VARIANTS.body1` styles.

export const Wrapper = styled(Typography).attrs({ type: 'body' })``;
